`$ARGUMENTS`를 파싱하여 Linux 운영 Runbook을 생성합니다.
- 첫 번째 인자: 대상 카테고리 (networking, storage, system, security)
- 나머지 인자: 작업명

## 파일 생성 규칙
1. 파일명: `runbook-linux-{작업명}.md`
2. 저장 위치: 해당 카테고리 디렉토리
3. `templates/runbook.md` 템플릿 기반으로 작성

## 작성 시 필수 포함 사항
- **사전 체크리스트**: 최소 4개 항목
- **환경 확인**: OS 버전, 커널 버전 확인 명령어
- **단계별 명령어**: 복붙 즉시 실행 가능
- **확인 명령어**: 각 스텝 완료 후 검증 방법
- **롤백 절차**: 원래 상태로 복구하는 명령어
- **위험 명령어 경고**: `> **주의**` 블록으로 명시

# Linux Docs

Linux 학습 문서는 운영 주제별로 나눠 관리합니다.

| 폴더 | 내용 |
|------|------|
| `system/` | 프로세스, 메모리, systemd, cgroup, 커널, 성능 진단 |
| `networking/` | TCP/IP, conntrack, iptables/nftables, tcpdump, eBPF/XDP |
| `storage/` | 파일시스템, LVM, mount, NFS/EFS, I/O, 패키지 관리 |
| `security/` | 계정, SSH, 권한, audit, SELinux, seccomp |
| `agents/` | AI 전문 에이전트 프롬프트 |
| `rules/` | 문서 작성과 Linux 운영 규칙 |
| `templates/` | 재사용 문서 템플릿 |

운영 자동화 자산은 `../ops/README.md`를 참고합니다.

`$ARGUMENTS`를 파싱합니다:
- 첫 번째 인자: 대상 파일 경로
- 나머지: 추가할 증상 설명

## 작성 형식

아래 형식으로 해당 파일의 트러블슈팅 섹션에 추가:

```markdown
### {증상명}

**증상**: {에러 메시지 또는 현상 설명}

**원인**: {근본 원인}

**해결 방법**:
```bash
# 해결 명령어
{명령어}
```

**재발 방지**:
- {예방 방법}
```

추가 완료 후 내용이 자연스럽게 이어지는지 확인하세요.
